# Value Node Template (Multi-language)

This template is available in multiple languages. Choose the version that suits your needs:

## English version
Located at: `locales/en/templates/value_node_template.md`

## Spanish version
Located at: `locales/es/templates/value_node_template.md`

## How to use
1. Choose your preferred language template
2. Copy it to `01_backlog/nodos/` with a proper node ID (e.g., `NV-001.md`)
3. Fill in the template according to your project needs

## Template structure
All templates follow this structure:
- **Objective**: Clear and measurable goal
- **Minimum context**: Essential information for execution
- **Input**: Required data, APIs, files, or decisions
- **Expected output**: Concrete deliverable
- **Acceptance criteria**: Validation requirements
- **Associated risk**: Risk assessment
- **Dependencies**: Related nodes or requirements
- **AI used**: Tools and models employed
- **Key human decision**: Critical human oversight
- **Status**: Current progress state

## Contributing new translations
To add a new language translation:
1. Create a new folder in `locales/` with the language code (e.g., `fr/` for French)
2. Add your translation in `locales/{code}/templates/value_node_template.md`
3. Update the CONTRIBUTING.md file with translation guidelines
